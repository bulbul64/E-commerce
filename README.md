# Personal Portfolio Website

This is my **personal portfolio website** built with **React, TypeScript, and Vite**.  
It showcases my projects, skills, and contact info in a modern, responsive layout.

🔗 Live Site: [Personal Portfolio](https://personal-portfolio-website-ten-liard.vercel.app/)

---

## 🖥 Features
- 🚀 Fast and optimized with **Vite**
- 🧠 Fully typed with **TypeScript**
- 🧰 Clean, responsive UI
- 📱 Works on all screen sizes (mobile, tablet, desktop)
- 🪄 ESLint integrated for code quality

---

## 🧠 Tech Stack
- **React**
- **TypeScript**
- **Vite**
- **JavaScript**
- **CSS / TailwindCSS (optional)**
- **ESLint**

---

## 📦 Dependencies
- `react`
- `react-dom`
- `vite`
- `@vitejs/plugin-react` or `@vitejs/plugin-react-swc`
- `eslint`
- `eslint-plugin-react-x`
- `eslint-plugin-react-dom`
- `typescript`
