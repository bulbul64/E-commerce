E-commerce Website

This is a simple e-commerce website built with React and Vite.
It is focused on learning frontend development and building UI-based shopping features.

🔗 Live Site: https://e-commerce-one-blush-82.vercel.app/

🖥 Features
🚀 Fast and optimized with Vite
🧠 Built with React (JavaScript)
🧰 Clean and responsive UI
📱 Works on all screen sizes (mobile, tablet, desktop)
🛒 Product listing layout
🪄 ESLint integrated for code quality
🧠 Tech Stack
React
JavaScript
Vite
CSS / TailwindCSS (optional)
ESLint
📦 Dependencies
react
react-dom
vite
@vitejs/plugin-react or @vitejs/plugin-react-swc
eslint
eslint-plugin-react-x
eslint-plugin-react-dom
🚀 How to Run Locally
git clone https://github.com/bulbul64/e-commerce-one-blush.git
cd e-commerce-one-blush
npm install
npm run dev
